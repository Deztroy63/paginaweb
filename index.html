Estructuras de Control

En Java, las estructuras de control permiten dirigir el flujo de ejecución del programa. A continuación, se presentan los bucles más comunes:

Bucle For


for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

El bucle for se utiliza para iterar un bloque de código un número específico de veces.

Bucle While


int i = 0;
while (i < 10) {
    System.out.println(i);
    i++;
}

El bucle while ejecuta un bloque de código mientras una condición sea verdadera.

Bucle Do-While


int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 10);

El bucle do-while garantiza que el bloque de código se ejecute al menos una vez.

Clases, Objetos y Métodos

La Programación Orientada a Objetos se basa en la creación de clases y objetos. A continuación, se presentan ejemplos básicos:

Definición de una Clase


public class Persona {
    String nombre;
    int edad;

    public void saludar() {
        System.out.println("Hola, mi nombre es " + nombre);
    }
}

Una clase Persona con atributos nombre y edad, y un método saludar.

Creación de un Objeto


Persona persona = new Persona();
persona.nombre = "Juan";
persona.edad = 30;
persona.saludar();

Se crea un objeto de la clase Persona y se llama al método saludar.
